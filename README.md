# SMK Kahuripan 1 Jakarta Website

This project is a basic website for SMK Kahuripan 1 Jakarta, built using Bootstrap. It includes a homepage with a header and a hero section, and is structured to allow for future complex development.

## Project Structure

```
smk-kahuripan-website
├── src
│   ├── assets
│   │   ├── css
│   │   │   ├── style.css
│   │   │   └── custom.css
│   │   ├── js
│   │   │   └── main.js
│   │   └── fonts
│   ├── components
│   │   ├── header.html
│   │   ├── footer.html
│   │   └── nav.html
│   └── pages
│       ├── index.html
│       ├── about.html
│       ├── academics.html
│       ├── facilities.html
│       ├── admission.html
│       └── contact.html
├── package.json
└── README.md
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd smk-kahuripan-website
   ```

3. Install dependencies:
   ```
   npm install
   ```

## Usage

To start the development server, run:
```
npm start
```

Open your browser and go to `http://localhost:3000` to view the website.
